08 — Built Things

Projects

AI 02
Artificial Intelligence (AI) & Language Assessment: Enhancing Access in Multilingual Children
This project explores using machine learning and speech-based NLP to analyze children’s speech in non-word repetition (NWR) tasks. NWR is a strong marker for detecting language impairment. The goal is to build an automated scoring system that aligns with human evaluation and supports language-neutral assessment for multilingual children.
Python Speech Recognition Natural Language Processing Machine Learning HuggingFace Wav2Vec
KG 01
A Fine-Grained Study of Long-Tail Entities in News Articles
This project investigates long-tail entities in news articles, which are often missing from traditional knowledge bases. Using fine-grained entity typing, we analyze how emerging entities (e.g., startups or local politicians) appear in news datasets. The goal is to improve entity recognition and classification in dynamic news streams.
Python Signal-1M Knowledge Graph GliNER Genre Natural Language Processing

AI 01
QRC vs DQN on a New Environment (Truck Backer Upper)
This project compares Deep Q-Learning (DQN) and Q-Learning with Regularized Corrections (QRC) in the Truck Backer Upper environment. We analyze their learning speed, stability, and performance across training runs. Results show that QRC improves early learning while DQN provides more stable performance across seeds.
Python Pytorch Reinforcement Learning DQN QRC
